# Buer i Norge – «Buboka»

Privat, passordbeskyttet oversikt over åpne buer, koier, gammer og gapahuker i den sørnorske fjellheimen – samlet fra fjellstyrer, kommuner, turlag, jakt- og fiskeforeninger og allmenninger. Buer på bookingtjenester (Inatur, Airbnb, finn.no o.l.) er bevisst utelatt.

Hele siden er klientside-kryptert (AES-256-GCM, PBKDF2 310k iterasjoner). Uten nøkkelordet er innholdet i dette repoet bare støy.

Datagrunnlag skrapt natt til 22. juli 2026. Posisjoner kan være omtrentlige – sjekk alltid kilden før du legger ut, og vis buvett.
